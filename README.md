    <h2>De onde vem o Pix</h2>
    <p>    HISTÓRIA DO PIX

Sistemas de pagamento em tempo real ou Real Time Payment, sua denominação internacional, existem desde a década de 1970 e o seu uso aumentou muito nos últimos anos. São Sistemas PIX:
1.Pix : Lançado em 2020 pelo Banco Central do Brasil, que permite pagamentos instantâneos  24 horas por dia e 7 dias por semana.
2.Faster Payment System (FPS) : Lançado em 2008, o FPS é um dos primeiros sistemas de pagamento em tempo real do Reino Unido. 
3. Immediate Payment Service (IMPS) : Lançado em 2010 pela National Payments Corporation of India (NPCI), o IMPS é um sistema de transferências eletrônicas entre bancos. 
4.Real-Time Payments Network by The Clearing House : O primeiro novo sistema de pagamentos central na América do Norte em mais de 40 anos. 
5.FedNow Service : Lançado pelo Federal Reserve em julho de 2023. 
6.PayNow : Lançado pela Monetary Authority of Singapore, o PayNow permite pagamentos em tempo real entre contas bancárias de bancos participantes. 
7.Swish : Uma app móvel que permite pagamentos em tempo real entre pessoas e entre pessoas e empresas. 
Os pagamentos em tempo real permitem a transmissão da mensagem de pagamento e a disponibilidade dos fundos quase em tempo real. 
Embora os sistemas de pagamento em tempo real existam de alguma forma desde a década de 1970, a adoção global do RTP aumentou drasticamente na última década.  
         </p>

    <h2>Como e por quê usar o Pix</h2>
    <p>    COMO FUNCIONA O PIX

Pagamentos em tempo real são pagamentos instantâneos processados de forma imediata e contínua, 24 horas por dia. O PIX transfere fundos imediatamente de uma conta bancária para outra. Isso pode ocorrer segundos após o início da transação e os fundos ficam imediatamente disponíveis para o destinatário. O processo começa quando o pagador decide enviar dinheiro ao beneficiário. O pagador pode iniciar isso em vários lugares, como plataformas bancárias online, aplicativos bancários móveis ou pessoalmente em um local físico. O pagador também pode usar dados tradicionais, como números de conta, ou métodos mais modernos, como a leitura de um código QR. 
Depois que o pagador inicia o pagamento, a próxima etapa é a autenticação e autorização. O banco do pagador autentica a identidade do pagador, geralmente por meio de medidas de segurança como senhas, dados biométricos ou autenticação de dois fatores. Em seguida, o banco verifica se o pagador tem fundos suficientes para a transação. Se tudo for aprovado, o banco autoriza a transação. 
Após o processamento do pagamento, tanto o pagador quanto o beneficiário recebem notificações sobre a transação. Essa notificação pode ser um SMS, um e-mail ou uma notificação do aplicativo bancário, dependendo de como o pagamento foi iniciado.          </p>

<script>
  const links = document.querySelectorAll('nav.menu-inferior a');
  const pages = document.querySelectorAll('.page');
  const relatosList = document.getElementById('relatosList');
  const form = document.getElementById('relatoForm');
  const formMsg = document.getElementById('formMsg');

  // Função para mostrar a página selecionada
  function showPage(id) {
    pages.forEach(p => p.classList.remove('active'));
    document.getElementById(id).classList.add('active');
    links.forEach(link => link.classList.remove('active'));
    document.querySelector(`nav.menu-inferior a[data-target="${id}"]`).classList.add('active');
    if(id === 'page5') {
      carregarRelatos();
    }
  }

  // Evento para navegação do menu
  links.forEach(link => {
    link.addEventListener('click', e => {
      e.preventDefault();
      const target = link.getAttribute('data-target');
      showPage(target);
    });
  });

  // Salvar relato no localStorage
  form.addEventListener('submit', e => {
    e.preventDefault();
    const nome = form.nome.value.trim();
    const email = form.email.value.trim();
    const relato = form.relato.value.trim();
    if(nome && email && relato) {
      let relatos = JSON.parse(localStorage.getItem('relatosPix')) || [];
      relatos.push({nome, email, relato, data: new Date().toLocaleString()});
      localStorage.setItem('relatosPix', JSON.stringify(relatos));
      form.reset();
      formMsg.style.display = 'block';
      setTimeout(() => formMsg.style.display = 'none', 3000);
    }
  });

  // Carregar relatos da aba 5
  function carregarRelatos() {
    relatosList.innerHTML = '';
    let relatos = JSON.parse(localStorage.getItem('relatosPix')) || [];
    if(relatos.length === 0) {
      relatosList.innerHTML = '<li>Nenhum relato registrado.</li>';
      return;
    }
    relatos.forEach(r => {
      const li = document.createElement('li');
      li.innerHTML = `<strong>${r.nome}</strong> (${r.data})<br/>${r.relato}`;
      relatosList.appendChild(li);
    });
  }
</script>
